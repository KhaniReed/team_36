Welcome! This tool will be your Phishing Detection Companion! Due to the rise in global phishing attacks it is necessary to have a tool that can detect phishing attacks.

This tool is designed to help you identify phishing websites and emails to help protect your sensitive information.

Before starting there is some information to know about this tool:
To get the best results, please ensure when entering URLs to have https:// or http:// at the beginning of the URL. In majority of cases, the tool will be able to detect phishing attacks without this, but it is best to have it.
Also for us to be 100% transparent with you we would like to inform you that this tool is not 100% accurate and should not be used as a sole source of truth. For email detection this tool provides about 98% accuracy rate and for URL detection it provides about 81% accuracy rate.

To get started, please select the Phishing Detection option from the navigation bar above.
If you would like to see who worked on this project and connect with us, please select the Developers option from the navigation bar above.

We hope you enjoy using our phishing detection tool and find it helpful in your phishing detection efforts!
